# Student List Clicker

Microsoft Access アプリケーション「School Leader - Ver. 6.9」から学生データを自動抽出するためのツールです。

## 概要

このツールは Rust の `uiautomation` クレートを使用し、Access 上の学生リストを自動的にクリックして巡回しながら、複数のタブに分散している情報を収集し、ひとつの CSV ファイルに統合します。

Access の不安定な UI 構造に対応するため、**「タブの切り替えは人間が行い、抽出はプログラムが自動で行う」**セミオート形式を採用しています。

## 準備

1.  **School Leader** を起動し、学生リストが表示されている画面を開いておきます。
2.  Rust がインストールされていることを確認してください。

## 使い方

1.  ターミナル（PowerShell等）でプロジェクトディレクトリに移動します。
2.  以下のコマンドでビルドと実行を行います：
    ```powershell
    cargo build
    .\target\debug\student_list_clicker.exe
    ```
3.  **抽出人数の指定**:
    `抽出する最大人数を入力してください (例: 49):` と表示されるので、処理したい人数を入力して Enter。
4.  **フェーズ 1 (基本情報)**:
    Access 側で「学生情報」タブを開き、ターミナルで Enter。自動巡回が始まります。
5.  **フェーズ 2 (通学経路)**:
    巡回が終わると一時停止します。Access 側で「通学経路/緊急連絡先」タブを開き、ターミナルで Enter。
6.  **フェーズ 3 (父母等)**:
    再び一時停止します。Access 側で「父母等/保証人/負担者」タブを開き、ターミナルで Enter。
7.  **完了**:
    すべてのフェーズが終わると、同ディレクトリに `student_data.txt` が生成されます。

## 出力項目

出力される CSV は以下のヘッダーを持ちます：
`学籍番号, 氏名, ふりがな, 英名, 生年月日(西暦), 生年月日(和暦), 国籍, 本人住所1, 本人住所2, 本人住所3, 本人TEL, 本人携帯, 本人メール1, 本人メール2, 本人通学路線, 本人最寄駅, 本人所要時間(分), 緊急氏名, 緊急ふりがな, 緊急続柄, 緊急電話, 保護者氏名, 保護者続柄, 保護者携帯, 保護者住所1, 保護者住所2, 保護者住所3`

## 注意事項

- **待機時間の調整**: Access は高速な自動操作に弱いため、デフォルトで 1 人あたり 800ms の待機時間を設けています。この時間は `config.toml` の `wait_ms` で変更するか、コマンドライン引数で指定できます：
    ```powershell
    .\target\debug\student_list_clicker.exe --wait-ms 1000
    ```
- **重複フィールド**: 氏名など、画面上に同じ名前のフィールドがある場合、より右側に配置されている要素（入力フィールド）を優先的に取得するロジックを搭載しています。
- **実行中の操作**: 自動巡回中にマウスやキーボードを操作すると、クリック位置がズレて失敗する可能性があるため、巡回中は PC を操作しないでください。
